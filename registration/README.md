# User Registration Form - Flutter Application

A complete user registration form with validation built with Flutter. This application demonstrates form handling, input validation, and user feedback through dialog boxes.

## 📱 Features

- **Form Validation**: Real-time validation for all input fields
- **Field Validations**:
  - Full Name: Required field, cannot be empty
  - Email: Required field, must contain '@' symbol
  - Password: Required field, minimum 6 characters
  - Confirm Password: Must match the password field
- **User Feedback**: Success dialog with user's name after successful registration
- **Clean UI**: Material Design with custom styling and icons
